# 💍 Lista de Presentes de Casamento

Este é um site divertido e responsivo para que os convidados possam presentear os noivos com itens selecionados, com pagamento via Stripe.

## 🔧 Tecnologias

- React + Vite
- Tailwind CSS
- Stripe Checkout
- Framer Motion
- Deploy no Vercel ou GitHub Pages
